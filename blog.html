Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents, which defines the logical structure of documents and how they are accessed and manipulated. In web development, the DOM represents the HTML or XML of a webpage as a tree of nodes, where each part of the document (elements, attributes, and text) is a node that can be interacted with using JavaScript.

How do you select an element from the DOM?

  • getElementById("id"): Selects an element by its id.
  • getElementsByClassName("class"): Selects all elements with a specific class.
  • getElementsByTagName("tag"): Selects all elements with a specific tag name.
  • querySelector("selector"): Selects the first element that matches a CSS selector.
  • querySelectorAll("selector"): Selects all elements that match a CSS selector.
  • parentNode: Selects the parent of an element.
  • children: Selects the child elements of a parent.
  • closest("selector"): Selects the nearest ancestor matching the selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the context of the DOM is a technique where a single event listener is attached to a parent element, and it monitors events triggered by its child elements. Instead of adding event listeners to each individual child, the event is "delegated" to a common ancestor, which handles events as they propagate upward through the DOM (event bubbling).

How do you manipulate an element's attributes and styles using the DOM?

  • getAttribute("attributeName"): Retrieves an attribute's value.
  • setAttribute("attributeName", "value"): Sets or updates an attribute.
  • removeAttribute("attributeName"): Removes an attribute.
  • element.style.property = "value": Modifies an element's CSS inline.
  • Example of changing color: element.style.color = "blue" changes the text color to blue.
  • Example of changing background: element.style.backgroundColor = "yellow" changes the background color.
  • element.classList.add("className"): Adds a class.
  • element.classList.remove("className"): Removes a class.
  • element.classList.contains("className"): Checks if a class exists.
